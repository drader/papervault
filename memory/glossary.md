# Glossary

Proje genelinde kullanılan terimler sözlüğü.
`npm run setup` sonrası intake interview cevaplarıyla doldurulur.

| Terim | Tanım |
|---|---|
| INGEST | Ham kaynak → wiki entegrasyonu |
| QUERY | Wiki'ye soru sorma + cevabı geri-dosyalama |
| LINT | Wiki sağlık kontrolü |
| CONSOLIDATE | MEMORY.md → wiki/concepts/ terfi operasyonu |
| Tier 1 | Her oturumda yüklenen bağlam (memory/, journal/, scratch/) |
| Tier 2 | İlgili alan aktifken yüklenen bağlam (wiki/, agent dosyaları) |
| Tier 3 | Sadece istenince yüklenen arşiv içerik |
